# MVP Validation Checklist: Schedule & Todo Management with AI

## US1: 基础日程与待办管理
- [x] 待办列表能够正确加载、添加、勾选状态和删除
- [x] 日程管理能够通过迷你日历切换日期，新建并正确显示时间段
- [x] 所有数据在页面刷新后依然正确显示 (MySQL 持久化)

## US2: AI 辅助待办草稿生成
- [x] 点击 AI 助手按钮可打开/关闭侧边栏
- [x] 输入自然语言指令，AI 能返回任务草稿
- [x] 点击“保存到待办”，任务能进入待办列表且刷新可用

## US3: AI 指令批量执行
- [x] 输入批量操作指令 (如“完成所有测试任务”)，显示确认弹窗
- [x] 确认弹窗显示正确的变更预览和任务列表
- [x] 点击“确认执行”后，待办列表状态同步更新

## US4: AI 自由对话与持久化 (新增)
- [x] 支持不属于任务或指令的自由对话
- [x] 对话历史自动保存到数据库
- [x] 刷新页面或重新进入可恢复对话上下文
- [x] 支持手动“清除”对话并开启新会话

## 异常流程与 UI
- [x] 后端服务不可用时，AI 助手有明确的错误提示
- [x] 列表为空时显示 Empty 状态
- [x] 数据加载时显示 Loading 状态
- [x] 侧边栏布局在折叠/展开时工作正常
